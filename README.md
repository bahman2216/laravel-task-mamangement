## Laravel task management

This is a simple package to create, edit, delete and sort tasks asigned to users. Using MySQL as default DB and laravel V.9

## Usage

First of all, create MySQL(or any other suitable database) and config .env file for database credentials.
Then enter below code in your terminal:
```
php artiasn migrate
```

Goto the http://localhost:8000/tasks or http:127.0.0.1:8000/tasks
